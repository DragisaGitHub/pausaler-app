# pausaler-app

A desktop application designed for **sole proprietors in Serbia (pausalci)** to simplify
invoicing, document management, and everyday administrative work.

The application is built as a **local desktop app** using React and Tauri, enabling fast
performance, offline usage, and full control over business data without reliance on cloud
services.

## Purpose

`pausaler-app` is intended for entrepreneurs operating under the Serbian lump-sum taxation
model who need a simple and reliable tool to manage:

- invoice creation and tracking
- storage and organization of business documents
- basic client and business records
- daily administrative tasks without complex accounting systems

## Key Features

- Desktop application (Windows / macOS / Linux)
- Local data storage
- Offline-first usage
- Clean and focused user interface
- Built around real-world needs of Serbian sole proprietors

## Technology Stack

- **React** – user interface
- **Vite** – frontend tooling
- **Tauri** – desktop application framework
- **TypeScript** – type safety and stability

## Project Status

- The core application is already implemented locally
- This repository serves as the central place for further development and maintenance
